# 👋🏼 Introducción

En esta web, encontrarás la documentación para poder colaborar en el proyecto, forma de trabajo y elaboración del trabajo práctico de Algortimos 2.

## Cómo usar esta documentación?

Para navegar por la documentación, puedes utilizar la barra lateral que se encuentra a la izquierda de la pantalla. En ella, encontrarás una lista de los temas que se tratan en esta documentación.

### Localmente

Si querés ver la documentación localmente, sigue los siguientes pasos:

1. Cloná el repositorio y movete a la carpeta `project-docs`:

```bash title="📂 /"
>> git clone https://github.com/JaviCeRodriguez/tp_algoritmos2.git
>> cd tp_algoritmos2/project-docs
```

2. Instalá las dependencias y ejecutá el servidor de desarrollo:

```bash title="📂 /tp_algoritmos2/project-docs"
>> npm install
>> npm run start
```

3. Abrí tu navegador y navegá a `http://localhost:3000/`. Ahí podés ver la documentación!

4. Si querés editar los archivos de la documentación, puedes hacerlo en la carpeta `project-docs/docs` e ir al documento que quieras editar.

5. Para detener el servidor, presiona `Ctrl + C` en la terminal.


## Cómo usar las Jupyter Notebooks?

:::tip Antes de avanzar

Tanto para Visual Studio Code como en Jupyter (Lab), vas a tener que utilizar un entorno virtual de Python:

```bash title="📂 /"
>> python -m venv venv
>> source venv/bin/activate # En Windows, es venv\Scripts\activate
(venv) >> pip install -r requirements.txt
````

:::
